Themen: * Docker als Development-Container Zeit: 20:30 Uhr Ort: Uni Leipzig Raum P-801, Augustusplatz 10, 04109 Leipzig, Deutschland [[Map]] [Map]: https://mapsengine.google.com/map/edit?mid=zeodHxxijHv8.krpQ0SXKhbEA **Update** Das waren die Themen: Docker.io * Erste Schritte * Dockerfile Befehle * Vagrant mit Docker-Provision Wichtige Befehle: * Dockerfile bauen ```sudo docker build --rm=true .``` * Docker Container starten ```sudo docker run -t -i IMAGENAME /bin/bash``` * Docker Container starten und Verzeichnis mounten ```sudo docker run -t -i -v $(pwd):/var/www IMAGENAME /bin/bash``` * Docker Container als Image speichern ```sudo docker commmit CONTAINERID REPONAME:TAGNAME``` * Docker Container exportieren ```sudo docker export CONTAINERID > NAME.tar``` * Docker Image exportieren ```sudo docker save -o FILENAME IMAGEID``` * Löschen aller Container ```sudo docker rm $(docker ps -a -q)``` * Löschen aller Images ```sudo docker rmi $(docker images -a -q)```
